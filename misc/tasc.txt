Проект "DelayGram"

Модальное окно:
Поиск картинок через модальные окна;
Всегда 9 картинок. Если часть не загрузилось, отправляем дополнительный запрос.
Есть кнопки следующие / предыдущие для текущего запроса.


Основное окно:
Перетаскивание меняет порядок;
Есть зона для удаления перетаскиванием;
Сохраняется время добавления и текст запроса (хидер и фитер для полного экрана);
Щелчек по картинке - вывод на полный экран, есть кнопки стрелок в стороны для перехода;
Все это добро хранится в куки;
Кэширование - повторных аналогичных запросов (без куки);

Лайтбокс:
C перемоткой влево\вправо.
